## Objetivo

Praticar o uso de strings e formatação de mensagens. Neste exercício, você irá trabalhar com entrada de dados e gerar uma mensagem personalizada.

## Especificação

### Mensagem de Boas-vindas

Abra o arquivo main.py. Dentro dele, localize a função resposta.

A função deverá receber um valor do tipo string representando o nome do usuário e retornar uma mensagem de boas-vindas personalizada.


Regras:

- Utilizar o nome recebido para montar a mensagem
- A mensagem deve conter uma saudação ao usuário
- Retornar a mensagem como uma string

Exemplos:

- `resposta("João")` → `"Bem-vindo, João!"`
- `resposta("Maria")` → `"Bem-vindo, Maria!"`

**Atenção:** utilize return, não print
